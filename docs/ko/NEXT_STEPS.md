# Session Handoff

## Canonical Source Files
- `NEXT_STEPS.md`

## Purpose
이 companion은 Codex session 간에 작업을 이어가기 위한 live handoff 문서를 설명합니다.

## Main Contents
- current pipeline status와 최근 verified output
- active log, model state, repair status
- immediate next steps와 recommended resume sequence
- context rollover 이후에도 유지되어야 하는 important decisions

## Practical Meaning
새 세션이 현재 engineering/ML 작업을 state 손실 없이 이어가야 할 때 가장 먼저 보는 resume 문서입니다.
