## 완료된 내용

- Google Cloud CLI 워크플로 지원을 통해 Borg 추적 샤드를 다운로드했습니다.
- 원시 Borg 추적 데이터를 클러스터별 parquet 출력으로 flatten했습니다.
- 결합된 사용량, 이벤트 및 기계 데이터세트를 구축했습니다.
- 15분 실패 라벨이 포함된 예측 데이터세트를 구축했습니다.
- 최초의 Polars 기반 예측 기준선을 훈련하고 평가했습니다.
- 클러스터 지표, 기능 순위, 최고 위험 경고를 포함한 기본 검사 아티팩트를 내보냈습니다.
- 시간적 특징을 추가하고 여러 특징 프로필을 평가했습니다.
- `base`을 가장 강력한 기본 프로필로 유지하고 경고 중심 대체 프로필로 `base_plus_roll`을 기록했습니다.
- Borg 파생 데이터 세트에 대한 정식 예측 스키마 내보내기가 추가되었습니다.
- parquet parquet 또는 CSV 원격 측정을 정식 예측 스키마에 매핑하는 로컬 클라우드 어댑터가 추가되었습니다.
- 평가 Result 및 스키마 참조를 위한 보고서 파일이 추가되었습니다.
- 강화된 커밋 샤딩 정책 및 지속적인 마일스톤 워크플로 규칙.

## 현재 운영 규칙

- 관심사와 파일 클래스별로 커밋을 공격적으로 분할합니다.
- `reports/` 형식으로 KST 타임스탬프가 앞에 붙은 파일 이름을 `YYYYMMDDHHMM_*` 형식으로 사용하세요.
- 세션을 종료하기 전에 추적된 메모리 파일에 현재 상태를 유지하라는 명령으로 `milestone`을 처리합니다.
